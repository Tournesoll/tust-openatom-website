## 🎯 用户使用流程（简化后）

1. **下载应用包** → 解压
2. **双击运行** → 自动打开向导
3. **按照向导操作**：
   - 创建 GitHub OAuth App（有详细教程）
   - 填写配置信息
   - 点击"开始部署"
4. **等待部署完成** → 获取网站地址
5. **开始使用** → 进入管理后台

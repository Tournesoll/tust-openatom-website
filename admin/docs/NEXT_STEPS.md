### 选项1：打包桌面应用（推荐优先）

**目标**：将应用打包成可执行文件，分发给用户使用

**步骤**：
```bash
cd admin
npm run build:win    # Windows 打包
npm run build:mac     # macOS 打包
npm run build:linux   # Linux 打包
```

**完成后**：
- 打包文件在 `dist/` 目录
- 可以直接分发给用户
- 用户无需安装 Node.js 即可使用

**预计时间**：10-20 分钟

欢迎使用 Jekyll 搭建你的第一个博客！

## 什么是 Jekyll？

Jekyll 是一个简单的静态站点生成器，特别适合用来搭建博客。它有以下特点：

- **简单**：使用 Markdown 编写文章
- **灵活**：支持自定义主题和插件
- **免费**：可以部署到 GitHub Pages
- **快速**：生成静态页面，访问速度快

## 如何写一篇文章？

1. 在 `_posts` 目录下创建新文件
2. 文件名格式：`YYYY-MM-DD-标题.md`
3. 在文件开头添加 Front Matter
4. 使用 Markdown 编写内容

## Front Matter 示例

```yaml
---
layout: post
title: "文章标题"
date: 2025-12-04 22:30:00 +0800
categories: 分类
tags: [标签1, 标签2]
---
```

## 常用 Markdown 语法

### 标题

使用 `#` 创建标题，一个 `#` 是一级标题，两个 `##` 是二级标题。

### 列表

- 列表项 1
- 列表项 2
- 列表项 3

### 代码

行内代码使用 \`代码\`。

代码块使用三个反引号：

```python
def hello():
    print("Hello, Jekyll!")
```

### 链接

[Jekyll 官网](https://jekyllrb.com/)

### 图片

```markdown
![图片描述](图片URL)
```

## 🎨 设计系统

### 配色方案（灰色系）
```css
/* 背景色 */
--bg-dark: #0a0a0a
--bg-main: #1a1a1a
--bg-secondary: #1F2937

/* 文字色 */
--text-primary: #F9FAFB
--text-secondary: #D1D5DB
--text-gray: #9CA3AF

/* 强调色 */
--accent: #6B7280
```

### 字体系统
- **标题**：超大字体（3-6rem），粗体（700）
- **副标题**：中等字体（1.25-2rem），常规（400）
- **正文**：标准字体（1rem），行高 1.6-1.8

### 间距系统
- **Section**：8rem 上下间距
- **卡片间距**：2rem
- **元素间距**：1-1.5rem

### 圆角规范
- **卡片**：24px
- **按钮**：980px（完全圆角）
- **标签**：100px

### 阴影效果
- **悬停阴影**：`0 20px 40px rgba(0, 0, 0, 0.3)`
- **按钮阴影**：`0 4px 14px rgba(107, 114, 128, 0.4)`
